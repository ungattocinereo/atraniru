# Atrani.ru Project

Website for atrani.ru — travel and tourism services on the Amalfi Coast, Italy.

## Project Skills

This project has two design skills for creating pages:

### `/atrani-static-page`
Use for: landing pages, about pages, services, apartments, contact — any non-blog content.

### `/atrani-blog-article`
Use for: blog posts, travel guides, stories, tips, reviews — editorial content with full article structure.

## Design System

Navy Brutalist (Concept 8) — dark navy backgrounds with orange accents.

**Key colors:**
- Navy deep: `#000A27`
- Navy: `#0A1628`
- Orange accent: `#FF5722`

**Fonts:** Unbounded (display) + Space Grotesk (body)

## Reference Files

- Static page template: `concept-8-navy-brutalist.html`
- Blog article template: `concept-8-blog-article.html`
- Concept gallery: `index.html`

## Language

All content is in Russian. Brand name: **АМАЛЬФИГЕТЬ** (wordplay on Amalfi + Russian expression).

Headline format: **АМАЛЬ-ФИГЕТЬ!** (with hyphen, split on two lines).
